# Lockout Sentinel

**Lockout Sentinel** is a VB.NET application designed to help IT administrators monitor and manage account lockouts in their environment. It allows you to receive alerts when users are locked out of Active Directory (AD) or Network Policy Server (NPS), and provides tools to unlock users quickly and safely.  

---

## Features

- Real-time notifications when users are locked out of AD or NPS  
- Ability to unlock AD users directly from the application  
- Unlock NPS users without needing to manually check logs  
- Easy-to-use VB.NET interface  
- Designed for IT admins to streamline account management  

---

## Requirements

- Windows OS  
- .NET Framework 4.7.2 or later  
- Administrative privileges for unlocking AD/NPS users  

---

## Installation

1. Clone or download the repository:  
```bash
git clone https://github.com/BrandonSanders48/Lockout-Sentinel-by-Brandon.git
````

2. Open the solution in Visual Studio.
3. Build the project (Debug or Release).
4. Run `LockoutSentinel.exe`.

---

## Usage

1. Launch the application.
2. Configure your AD and NPS connection settings.
3. Start monitoring to receive real-time alerts for user lockouts.
4. Use the built-in unlock feature to quickly restore access for locked-out users.
